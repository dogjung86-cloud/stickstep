# 역사① Ⅳ(h1u4) 발주 — 스틱맨 개념 컷 10장(public/his/cuts) + 만화 3편 21컷
#   (public/comics/h1u4l3 · h1u4l4 · h1u4l8, 각 7컷)
# 컷·만화 모두 "이미지 안 글자 절대 금지". 만화 컷은 앱이 한글 말풍선을 얹는 하이브리드라
# 인물이 입을 벌리고 손짓하는 "말하는 연기"만 그리고, 인물 머리 위쪽에 여백을 넉넉히 남긴다.
# 표시 프레임이 정사각(1:1 cover 크롭)이므로 주인공은 가운데 배치, 좌우 가장자리엔 핵심 요소 금지.
# ★ 코믹 격상 표준(h1u2·h1u3 계승): 코믹 컷은 "GAG:" 문장으로 무엇이 웃긴지 명시, 죽음·전쟁·정복
#   컷은 DIGNIFIED 표기(개그 금지). 실존 인물(쿠빌라이 칸·영락제·정화·메흐메트 2세)은 존엄 유지 —
#   개그는 주변 인물·동물(기린) 몫.
# ★★ 정복·수탈 존중 가드(Ⅳ 제1원칙): 전투·유혈·죽음 장면은 그리지 않는다 — 1453 함락 컷은 성문과
#   깃발·행렬로만 상징. 종교 존중 가드(Ⅲ 계승): 종교 창시자와 신앙 대상(무함마드·부처·예수·힌두 신
#   등)은 어떤 형태로도 그리지 않는다. 사원·모스크는 건물 외관(돔·미너렛·탑)만.
#
# ── 스타일 블록 A (스틱맨 개념 컷 공용 — his1·his2·his3와 동일) ──
# hand-drawn stick figure comic panel, black ink on clean white paper, thin uniform line weight,
# xkcd / whiteboard-marker doodle style, minimalist line art, no shading, no gradients, flat,
# charming educational. The recurring stick-figure character: plain round circle head, simple dot
# eyes, stick arms and legs, exactly five fingers where hands are drawn. Exactly one restrained
# accent color, teal, used only where the prompt says; everything else pure black and white.
# absolutely no text, no letters, no numbers, no words, no speech bubbles, no captions, no labels,
# no symbols anywhere in the image. NOT 3D, NOT photorealistic, NOT glossy render, NOT digital
# painting, NOT anime. Landscape 4:3, single panel, generous white space.
#
# ── 스타일 블록 C (만화 컷 공용 — 말풍선 하이브리드 + 코믹 연기) ──
# Hand-drawn stick figure comic panel, black ink lines on warm off-white paper,
# exactly one accent color: teal, used sparingly where the prompt says.
# Absolutely NO text, NO letters, NO numbers, NO speech bubbles, NO captions anywhere.
# Characters ACT like they are talking: open mouth (small oval), expressive hand gestures.
# Comedic acting is allowed and encouraged where the prompt marks GAG: exaggerated poses,
# big reactions, falling over backwards, sweat drops, bulging wide eyes, jaw dropping —
# but all historical objects, clothing and facts must stay accurate; never make the scene
# absurd beyond what the prompt describes, and NO anachronistic props (no phones, no modern
# items) inside historical scenes.
# CONQUEST RULE (absolute): never draw battle, blood, weapons striking people, or death.
# Conquest scenes are shown only through gates, flags, marching lines and architecture.
# RELIGION RULE (absolute): never depict founders or objects of worship of any religion —
# no Muhammad in any form whatsoever, no Buddha statues or Buddha figures, no Jesus, no Hindu
# deities. Religious buildings appear as exteriors only (domes, minarets, towers).
# Keep the upper third of the panel mostly empty white space above the characters' heads
# (the app overlays Korean speech bubbles there). Main subjects centered horizontally.
# Simple, charming, educational middle-school history comic. 4:3 landscape panel.
