# 중2 과학 VI. 동물과 에너지 — 이미지 발주안 A
# 스틱맨 개념 컷 6장

아래 명세에 따라 다운로드 가능한 독립 이미지 6개를 생성해 주세요.

## 작업 방식

- A-01부터 A-06까지 순서대로 각각 별도의 이미지로 생성합니다.
- 6칸짜리 콜라주, 컨택트 시트, 만화 페이지로 합치지 마세요.
- 한 이미지에는 지정된 장면 하나만 넣으세요.
- 여섯 장 모두 동일한 캐릭터 디자인, 선 굵기, 여백, 붉은 포인트 색을 유지하세요.
- 이미지 안에는 파일명, 제목, 설명을 포함한 어떤 글자도 넣지 마세요.
- 파일명은 이미지 바깥의 응답 텍스트로만 표시하세요.
- 각 이미지는 가로 4:3 비율로 생성하세요.
- 중간에 확인 질문을 하지 말고 6개를 모두 생성하세요.
- 한 응답에서 생성 가능한 이미지 수에 제한이 있으면 가능한 만큼 생성한 뒤, 다음 응답에서 완료되지 않은 번호부터 계속하세요.
- 이미 완료한 번호를 중복 생성하지 마세요.
- 마지막에 A-01~A-06 완료 체크리스트와 대응 파일명을 적어 주세요.

## 공통 시각 스타일

Hand-drawn stick figure comic panel in black ink on clean white paper. Thin, smooth, uniform line weight. Minimalist whiteboard-marker doodle aesthetic, flat and charming educational illustration. No shading, no gradients, no texture, and no realistic rendering.

The same recurring stick-figure character must appear throughout the set: plain circular head, simple dot eyes, expressive eyebrows when required, stick arms and legs, and exactly five fingers wherever a hand is clearly drawn.

Use exactly one restrained accent color: warm crimson red, approximately #C92A3A. Use it only on the element explicitly designated in each scene. Everything else must be pure black and white.

Absolutely no written text, letters, numbers, words, captions, labels, speech bubbles, logos, watermarks, or typographic symbols. Simple pictorial motion marks, airflow lines, heartbeat marks, sparkles, thought dots, and arrows are allowed only when explicitly requested by the scene description.

Not 3D, not photorealistic, not glossy, not digitally painted, not anime, and not a detailed cartoon. Landscape 4:3, single panel, centered subject, generous white space, clear silhouette, mobile-screen readability.

## 최종 자체 점검

생성한 각 이미지에 대해 다음을 확인하세요.

1. 독립 이미지 6개이며 콜라주가 아닌가?
2. 전부 가로 4:3인가?
3. 동일한 스틱맨과 동일한 선 스타일을 유지했는가?
4. 지정된 요소 하나만 따뜻한 진홍색인가?
5. 이미지 내부에 글자, 숫자, 파일명, 로고, 워터마크가 전혀 없는가?
