# 중2 과학 V 식물과 에너지 이미지 재생성 기록

현재 사용하는 10개 WebP의 **용도와 생성 의도 요약**이다. 이미지 생성 당시의 실제 프롬프트 원문은 이 저장소에 남아 있지 않으므로, 아래 문장은 원문을 복원하거나 새 프롬프트라고 주장하지 않는다.

## 공통 재생성 원칙

- 이미지 안에는 글자, 숫자, 수식, 화살표, 라벨을 넣지 않는다. 정확한 과학 정보는 HTML·SVG·Canvas 오버레이로 얹는다.
- 인물 컷은 기존 앱의 고급 스틱맨 만화 문법을 따르고, 장면의 핵심 행동과 표정이 작은 화면에서도 즉시 읽혀야 한다.
- 실사풍 그림은 깨끗한 조명, 명확한 주 피사체, 넉넉한 여백을 유지한다. 기공·엽록체·물관·체관처럼 판정이 필요한 구조를 생성 이미지에 맡기지 않는다.
- 최종 파일은 WebP로 만들고 긴 변 960px 이하를 지킨다. 품질 68~74를 출발점으로 하되 경계와 얼굴이 무너지지 않는 선에서 압축한다.
- 장당 120KB 아래를 우선하며, 현재 묶음처럼 30~75KB 수준을 목표로 한다. 투명도가 필요하지 않으면 불필요한 알파 채널을 제거한다.

## 배경·문제 그림 4종

| 파일 | 용도 | 생성 의도 요약 |
|---|---|---|
| `public/plant/figs/whole-plant.webp` | `sugarJourneyLab`의 식물 전체 베이스 | 뿌리부터 줄기, 어린잎, 꽃, 열매까지 한 화면에 들어오는 건강한 식물. 물관·체관과 양분 화물은 코드로 덧그릴 수 있도록 몸체 주변에 여백을 둔다. |
| `public/plant/figs/greenhouse.webp` | `photoFactorLab`의 스마트 온실 배경 | 센서와 재배 장치가 보이는 밝고 정돈된 스마트 온실. 화면과 계기판에는 읽을 수 있는 글자를 넣지 않고 그래프와 수치는 코드로 얹는다. |
| `public/plant/figs/day-night.webp` | 낮과 밤 비교 그림 | 같은 화분과 같은 구도를 낮·밤으로 나눈 장면. 기체 방향이나 광합성·호흡 라벨 없이 시간대와 빛의 차이만 선명하게 보인다. |
| `public/plant/figs/storage-foods.webp` | 광합성산물 저장 형태 문제 그림 | 포도, 사탕수수, 고구마, 콩, 깨가 서로 가리지 않고 한 화면에서 식별되는 정물. 포도당·설탕·녹말·단백질·지방 판정 문구는 앱이 제공한다. |
