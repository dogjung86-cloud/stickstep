## 확정된 방향 (사용자 확정 2026-07-12)

1차 개편 묶음 = **A+B 격상판**:
- **발자국 트레일**: 걸어온 구간의 연결선을 잉크 발자국으로(갈 길은 옅게).
  serpentine 경로 좌표를 그대로 샘플링해 도장 — 목업의 stamp() 함수가 구현 참고.
- **스틱맨 워커**: 현재 노드 위에 내 아바타 스틱맨(깃발). 레슨 완료 귀환 시 다음 노드까지
  걷기(8fps 두 포즈 플립북, 스플래시 125ms 리듬) + 발자국 스탬핑 + 깃발 플랜팅.
  **품질 게이트: 발주본(스플래시·만화) 옆에 놓고 위화감 없을 것 — 통과 전엔 지도에 안 올린다.**
  게이트가 늦으면 발자국(A)만 먼저 출시 가능(캐릭터 리스크 0).
- **눌리는 3D 스텝 노드**(듀오링고식): 노드에 아랫단(엣지) + 프레스 시 윗면 6px 침하.
  저빈도(레슨 진입)라 "고빈도 바운스 금지" 모션 규칙과 충돌 없음. 햅틱 탭(8)과 세트.
- **대단원 탭 = 2글자 키워드 필**(로마숫자 보류 — 목차/시험범위 톤이라 부담, 사용자 피드백):
  과학 중1 = 탐구·생물·열·물질·힘·기체·우주. 전체 제목은 파란 히어로 카드가 소유(중복 해소).
- **앱바**: 그리드 아이콘 → "과학 ▾" 과목 칩(현재 과목 표시 + 과목 허브 진입 겸용).
  우상단 프로필 버튼은 마이 탭(goTab("my"))으로 통일(현재는 로그인 화면으로 가는 개편 전 잔재).

보류/별도 결정: C(징검다리 리스킨 — '밟는' 감각은 노드 버튼화가 절반 흡수), D(공책 두들 전면 리브랜딩).
