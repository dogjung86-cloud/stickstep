# 스틱쌤(AI 튜터) 수익화 설계 — 세션 핸드오프 (2026-07-15)

앱 코드가 아니라 **수익화 결정 기록**이다. 다음 세션이 이 문서만 읽고 구현을 이어받을 수 있게 쓴다.
전제 기능(튜터 프로토타입·프리미엄 게이트)은 이미 구현·푸시됨 — CLAUDE.md "질문하기 — AI 튜터" 섹션 참조.

## 0. 결정 요약 (사용자 방향 승인 2026-07-15)

- **AI 튜터는 평생 이용권(프리미엄)에 무제한으로 묶지 않는다.** 콘텐츠는 한계비용 0이라 "평생 소장"이
  성립하지만, 튜터는 질문마다 원가가 나가는 종량 상품 — 평생 무제한 포함 = 원가 폭탄.
- 판매 구조 = **기간제 패스(주력) + 질문팩(보조) + 프리미엄 보너스 쿼터(맛보기)**. 아래 표.
- 구조는 승인, **가격 숫자는 제안값(안)** — 베타 사용량 실측 후 확정한다(6장 로깅 항목).

## 1. 원가 실측 (2026-07-15 기준 — 재검산 가능하게 근거 포함)

- 모델: Gemini 3.5 Flash, $1.50(입력)/$9(출력) per 1M tokens. 환율 ~1,400원/$ 가정.
- 질문 1건 구성(core/tutor.ts 현행): 시스템 프롬프트 ~1.3k + 오답 그라운딩 ~0.5k + 대화 누적
  + 답변 ~400tok(maxOutputTokens 1024 캡). 사진은 건당 +약 300~600tok.
- **질문당 원가: 텍스트 10~15원, 사진 포함 15~20원.** 대화가 길수록(히스토리 재전송) 턴당 상승.
  보수 설계값은 **건당 20원**.
- 시나리오: 월 100질문(현실적 평균) ≈ 2,000원 · 월 200질문 ≈ 4,000원 · 하루 20×30일 = 600질문 ≈ 12,000원.
- 원가 절감 여지(구현 레버, 5장): 프롬프트 캐싱 + 이미지 히스토리 최적화 + 모델 라우팅으로 30~50%↓ 추정.

## 2. 상품 구조 (구조 승인 / 가격은 안)

| 상품 | 가격(안) | 내용 | 노림수 |
|---|---|---|---|
| 스틱쌤 패스 30일 | **5,900원** (4,900~6,900 검토 범위) | **하루 20질문**, 수동 연장 — 자동결제 없음 | 주력. 시험기간 단위 구매 |
| 질문팩 30개 | **2,900원**, 유효 90일 | 충전형(소모성) | 라이트 유저·첫 결제 장벽 낮추기 |
| 프리미엄 보너스 | 무료 | 웰컴 30질문(1회) + **하루 첫 1질문 무료** | 매일 맛보기 → 패스 전환 훅 |

설계 근거(위반하면 안 되는 것):
- **"무제한" 표기 금지 — "하루 20질문" 정직 표기.** 중학생 체감상 무제한이면서 원가 최악 케이스가
  월 ~9,000원(20원×600)으로 캡. 무제한 표기+약관 제한보다 신뢰·법무 양쪽에서 깔끔. 악용(계정 공유·
  스크립트)도 자연 방어.
- **자동결제(정기구독) 금지.** 페이월 v3 신뢰 축이 "평생 소장·자동결제 없음" — AI만 자동구독이면
  약속이 깨진다. 수동 연장 30일권(한국 학부모에게 익숙한 이용권 충전 문법).
- **패스는 프리미엄 위의 애드온**(독립 판매 X). 튜터 가치의 원천(오답노트·레슨 그라운딩)이 프리미엄
  영역이라 독립 판매는 반쪽 경험. 2층 구조: 프리미엄(콘텐츠, 일시불) → 패스(AI, 기간제).
- **가격 사다리**: 팩 질문당 ~97원 vs 패스 액면 ~10원 → 패스가 압도적으로 이득으로 보이는 미끼 구조.
  실마진은 패스 유저 대부분이 쿼터를 다 안 쓰는 데서 나온다(월 100~200질문 → 마진 30~65%).
- **시장 앵커**: 콴다 프리미엄 월 18,500원(AI 질문 무제한 포함,
  https://qandahelp.qanda.ai/hc/ko/articles/15187595321625 — 2026-07-15 확인). 패스 5,900원 = 그 1/3.
  포지셔닝 문구 방향: "학원에서 질문 한 번 못 하는 값으로, 내 오답노트를 다 아는 과외쌤".
