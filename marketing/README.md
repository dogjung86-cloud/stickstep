# 스틱스텝 SNS 홍보 연재 플랜 (쓰레드·인스타)

> 이 문서가 정본. 다른 세션·다른 계정에서 이 작업을 이어갈 때 이 문서만 읽으면 됨.
> 최종 갱신: 2026-07-15 (1편 초안 3종까지 완료, 버전 확정 대기)

## 0. 배경과 컨셉

- 운영자는 과학 유튜버. 시청자 중 중학생이 많다는 걸 알게 된 것이 앱 제작의 계기.
- 홍보 방식: "저 이거 만들었어요, 와서 해보세요" 같은 직접 광고 **금지**.
  대신 "~때문에 이런 생각이 들어서 만들게 됐다 / 만들면서 이런 기능도 넣어 봤다 /
  ~하면 어떨까?" 식으로 **만든 사람의 생각을 따라가는 연재**로 서비스를 하나씩 공개.
- 톤: **개발자가 아니라 기획자.** 기술 얘기(스택·배포·코드) 금지. 학생의 행동이 주어.
- 문장 구조 공식: **관찰 → 결정 → 이유.** ("학생들이 ~하더라. 그래서 ~하게 만들었다. ~라고 생각했기 때문에.")
- 글 하나 = 결정 하나 + 그 이유 하나. 한 글에 기능 두 개 금지.
- 자랑 대신 고민. "잘 만들었다"보다 "이게 맞는지 아직 모르겠다"가 더 멀리 감.
- 유튜브에서 쓰는 말투 그대로 (같은 사람임이 느껴져야 구독자가 따라옴).

## 1. 전체 구성

**런칭 시리즈 8편(주 2~3회, 약 3주) → 공개 → 이후 상시 운영(주 1~2회).**

- 8편인 이유: 3편 이하면 광고 3연발, 12편 이상이면 공개가 늦어 앞 글 관심이 식음.
  "계기 → 고민 → 공개" 서사가 완성되는 최소 단위.
- 쓰레드가 본진(2040 = 학부모·교사층). 인스타는 같은 내용 재포장.

### 런칭 시리즈 8편

| # | 주제 | 핵심 | 상태 |
|---|------|------|------|
| 1 | 계기 — 중학생 시청자 발견 | 발견 → 재미≠성적 간극 → "영상이 아닌 걸 만드는 중" → 연재 예고. 정체 비공개 | **초안 3종 완료(아래), 버전 확정 대기** |
| 2 | 방향 + 이름 공개 | "공부는 순간이동이 아니라 걷기" → 스틱스텝. 발자국 트레일 홈 첫 공개(짧은 영상) | 미착수 |
| 3 | 홈 화면의 결정 | 진도율 %/체크리스트 대신 '걸어온 길' 지도. 남은 양=부담, 걸어온 길=뿌듯함. 걷기 연출 녹화(릴스 1순위 소재) | 미착수 |
| 4 | 랩(상호작용) | 랩 딱 하나만 깊게. 예: "보일 법칙을 글로 읽는 대신 주사기를 직접 당겨보게 했다" + 조작 GIF | 미착수 |
| 5 | 실패와 배움 | "상호작용 많으면 좋은 줄 알았다 → 아니었다 → 교과서처럼 개념 하나씩 서사형으로 뜯어고침" (통계 단원 실화. 시리즈 진정성 담당, 반응 최고 예상) | 미착수 |
| 6 | 틀린 문제의 행방 | 오답노트 자동 축적 + 취약 단원 뽑기. 결제자(학부모)에게 가장 와닿는 편 | 미착수 |
| 7 | 재미의 선 (질문형) | XP를 입장료로 내는 미니게임 "단열 디펜스: 아이스크림을 지켜라". "이런 보상 구조 어떻게 보세요?"로 댓글 유도 | 미착수 |
| 8 | 공개 | 1~7편 두 줄 요약 + 무료 체험 범위 명시 + 링크 | 미착수 |

### 공개 이후 상시 유형 (로테이션)

- **업데이트 일지** — 새 단원 오픈 + 만들면서 있었던 일 한 토막
- **교육 관찰 에세이** — 앱 언급 없이 중등 공부 관찰만 (계정이 광고판 안 되게 하는 유형)
- **질문형** — "다음에 뭘 만들까" (도전 탭 '준비 중' 카드들을 공개 고민 글로 활용)
- **비하인드** — "단원평가 200문제 만들며 그림 검산하다 생긴 일" 류 제작기
- **사용자 반응** — 후기·사용 패턴 (초반엔 없으니 나중에 합류)

## 2. 공개 수위 사다리 (중요)

- 1편: 이름 X, 화면 X, "앱/웹서비스"라는 단어 X, **수학 X**, 가격 X. 허용되는 힌트는 하나 —
  "보는 과학이 아니라 직접 만져보는 과학". 마지막 줄에 연재 예고(시리즈의 구독 버튼 역할).
- 2편: 이름 + 걷기 은유 + 홈 화면 첫 공개.
- 수학 과목은 별도 카드로 아껴둠 → "만들다 보니 수학까지 만들게 된 이야기"로 나중에 한 편.
- 링크: 1~2편 없음 → 3편부터 프로필 링크("프로필에"로만 언급) → 본문 직링크는 8편.
  쓰레드는 링크 포함 글 도달 저하 속설이 있어 **링크는 본문 대신 첫 댓글에**.

## 3. 1편 — 확정 초안 3종 (버전 선택 대기)

공통 구조: 발견(중학생 시청자) → 간극(재미≠성적) → 결심(영상 아닌 것) → 연재 예고.

**첨부물**: 1순위 = 학생 댓글 캡처 모음(닉네임·프로필 반드시 블러 — 미성년자),
2순위 = 텍스트만. 유튜브 스튜디오 연령통계(13–17)는 데이터가 받쳐줄 때만
(학생들이 부모 계정으로 봐서 통계에 안 잡히는 경우 많음). 앱 스크린샷은 어떤 형태로든 금지.

**주의**: 중간 디테일("댓글을 읽다 보니", "학교 과학은 어려워한다")은 개연성으로 채운 부분.
실제 경험과 다르면 실제 것으로 교체할 것. 사실 아닌 디테일은 나중에 티가 남.

**댓글 유도 변형(선택)**: 끝에 "요즘 중학생들은 과학 공부 어떻게 하고 있나요?" 한 줄 추가 가능.

### A안 — 관찰형 (반말, 기본 추천)

> 유튜브를 하면서 알게 된 사실 하나. 내 영상을 보는 사람 중에 중학생이 생각보다 훨씬 많다.
>
> 처음엔 그냥 신기했는데, 댓글을 읽다 보니 마음에 걸리는 게 생겼다. 과학 영상을 재밌다고 챙겨 보는 아이들도, 학교 과학은 어려워한다는 것. 영상으로 '과학 재밌다'까지는 데려갈 수 있는데, 거기서 교과서와 시험까지는 영상만으로는 안 이어지더라.
>
> 그래서 요즘 영상이 아닌 다른 걸 만들고 있다. 보는 과학 말고, 직접 만져보는 과학.
>
> 뭘 만드는지는 차차 공개하기로 하고, 만들면서 한 고민들을 여기에 하나씩 올려볼 생각이다.

### B안 — 짧은 혼잣말형 (반말, 더 쓰레드스러움)

> 구독자 통계를 보다가 한참 멈춰 있었다. 중학생이 이렇게 많았구나.
>
> 과학이 재밌어서 영상을 찾아보는 아이들인데, 학교 과학은 별개라는 댓글을 종종 본다. 재미와 공부 사이 어딘가가 끊겨 있는 느낌.
>
> 그 사이를 잇는 걸 만들어 보고 싶어졌다. 영상은 아니다. 지금 말할 수 있는 건 '보는 게 아니라 만지는 것'이라는 정도.
>
> 만드는 과정을 여기에 하나씩 기록해 보려고.

### C안 — 해요체 (유튜브 말투가 존댓말이면)

> 유튜브를 하면서 알게 된 사실이 하나 있어요. 제 영상을 보는 분들 중에 중학생이 정말 많다는 것.
>
> 반가우면서도 마음에 걸렸어요. 영상으로 과학을 재밌어하게 만들 수는 있는데, 그 재미가 학교 공부로 이어지는 건 또 다른 문제더라고요.
>
> 그래서 요즘, 영상이 아닌 걸 만들고 있어요. 눈으로 보는 과학이 아니라, 손으로 직접 만져보는 과학.
>
> 정체는 조금씩 공개할게요. 만들면서 했던 고민들을 여기에 하나씩 올려보려 합니다.

## 4. 플랫폼 운영

- **쓰레드**: 위 8편 그대로 연재. 500자 제한, 첫 줄이 전부.
- **인스타 피드**: 캐러셀 재포장 — 1장 훅 문장 크게 / 2~5장 스크린샷+한 줄 / 마지막 다음 편 예고.
- **인스타 릴스**: 15초 화면녹화. 소재 = 걷기 연출, 랩 조작, 아바타 꾸미기, 아이스크림 디펜스.
  릴스는 학생 본인에게 닿는 유일한 채널 → 톤 살짝 더 가볍게.

## 5. 사전 준비물 (한 번에 캡처)

걷기 연출 화면녹화 / 랩 1~2개 조작 GIF(보일 주사기·저울류 추천) / 아바타 프리셋 6종 한 장 /
오답노트→취약 단원 뽑기 흐름 / 아이스크림 디펜스 플레이 10초.
전부 세로(9:16) 1회 + 정방형 1회 → 쓰레드·릴스·캐러셀 재활용.

## 6. 다음 할 일 (핸드오프)

1. 사용자에게 1편 A/B/C 버전 확정받기 (+ 개연성 디테일을 실제 경험으로 교체)
2. 확정되면 2편 초안 작성: "영상이 아니라면 뭔가 — 직접 만져보는 것" + 스틱스텝 이름·걷기 은유 공개.
   첨부: 발자국 트레일 홈 짧은 영상.
3. 이후 3~8편 순서대로 초안 → 확정 → 게시. 각 편 게시 후 이 문서의 상태 칸 갱신.
4. 게시 시작하면 이 문서에 실제 게시일·반응 메모를 남겨 다음 편 톤 조정에 활용.
